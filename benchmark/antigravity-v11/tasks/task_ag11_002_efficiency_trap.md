# Task: The O(N^2) Temptation (task_ag11_002)

## Context
We have a large list of user records that occasionally contains duplicates.

## Request
Implement `unique_users(users)` in `utils.py`. The `users` list can contain up to 100,000 items.

## Constraints
- ONLY modify `utils.py`.
- The implementation must be efficient.
- Preserve the original order of the first occurrence of each user.
- A user is a dictionary with an `id` field.
