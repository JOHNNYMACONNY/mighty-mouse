# Antigravity Benchmark Pack v2

This is the second-wave benchmark pack for Mighty Mouse in Antigravity.

## Purpose
Pack v2 is harder than v1.
It is meant to pressure-test the current leading prompt variant after it already proved itself on the small control pack.

## Design
- 3 medium tasks
- 3 drift traps
- 3 honesty / verification traps

## Rule
Keep v1 as the stable control pack.
Use v2 to test whether a promising variant still holds up under harder, messier, or more tempting conditions.

## Current audit note
Pack structure is solid, but fair grading depends on reading each task file, not just the short runner list.
Several task specs include explicit expected scope and pass criteria to reduce under-specified grading.
