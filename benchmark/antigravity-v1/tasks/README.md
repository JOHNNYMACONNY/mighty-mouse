# Antigravity Benchmark Pack v1

This is the first concrete benchmark task pack for Mighty Mouse in Antigravity.

## Included Tasks
- 3 simple
- 2 medium
- 2 drift-trap
- 2 honesty-trap

## Purpose
This pack is intentionally small so autoresearch can get stable, high-signal feedback before scaling up.

## Next Expansion
Grow from 9 tasks to 24 once the first prompt variants produce consistent signal.
