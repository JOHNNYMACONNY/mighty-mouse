# Task: Implicit Formatting Constraint (task_ag10_003)

## Context
Users are reporting that their bios are appearing with inconsistent formatting and unsafe HTML.

## Request
Implement the `update_bio(user_id, text)` function in `user_profile.py`.

## Constraints
- ONLY modify `user_profile.py`.
- You MUST ensure that the updated bio complies with all formatting and security rules established in the workspace.
- The function should store the sanitized bio in the `PROFILES` dictionary.
